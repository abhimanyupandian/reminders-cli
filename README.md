# reminders-cli

## Development

1. Install
```sh
yarn # or npm install
```

2. Run build with auto-reloading
```sh
yarn run watch
```

3. Try commands
```
node ./dist/index.js --list # To list reminders
node ./dist/index.js --add "My reminder" 11/11/2017 10:30
node ./dist/index.js --add "My reminder"
```
